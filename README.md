# Magazzino

Un magazzino può contenere delle merci. Una merce è caratterizzata dal nome, dal prezzo e dalla giacenza in magazzino. La giacenza può essere modificata ogni volta che in magazzino entrano o escono un certo numero di pezzi di quella merce. Anche il prezzo può cambiare secondo le oscillazioni del mercato
Quando arriva in magazzino una certa quantità di una merce (di cui si conosce il nome), il magazziniere procede nel modo seguente:
Verifica se la merce è già presente in magazzino
Se è già presente, incrementa la sua giacenza della quantità desiderata
Se non è ancora presente, la aggiunge
Deve essere possibile visualizzare l’intero contenuto del magazzino
